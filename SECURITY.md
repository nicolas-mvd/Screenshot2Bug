# Security

Screenshot2Bug stores user settings, drafts, screenshots, recordings, captured request evidence, optional OpenAI API keys, and optional GitHub OAuth tokens in local Chrome extension storage.

Do not commit API keys, GitHub tokens, exported bug report ZIPs, screenshots, recordings, captured request payloads, `.env` files, or other local capture artifacts to the repository.

If you find a security issue, please open a private report through GitHub security advisories for the repository owner.
