# Screenshot2Bug Privacy Policy

Last updated: May 23, 2026

Screenshot2Bug is a Chrome extension that helps users capture screenshots or videos of browser bugs and turn them into structured bug reports.

## Data The Extension Handles

Screenshot2Bug may handle the following data when you use the extension:

- Page information from the active tab, such as page title, URL, viewport, browser details, and screen details.
- Screenshots and video recordings that you choose to capture.
- Console errors and network request metadata from pages you choose to report.
- Reproduction steps, notes, and generated bug report text that you enter or create.
- Extension settings saved locally in Chrome, including optional OpenAI API keys, GitHub OAuth tokens, GitHub repository settings, Cloudinary cloud name, and Cloudinary upload preset.

## How Data Is Used

Screenshot2Bug uses this data only to provide its bug reporting features:

- Creating local draft bug reports.
- Generating Markdown bug report templates.
- Exporting ZIP files that contain report data and captured evidence.
- Optionally generating AI-assisted reports through OpenAI when you provide your own API key.
- Optionally creating GitHub issues when you connect GitHub.
- Optionally uploading screenshots to Cloudinary before embedding them in GitHub issues.

## Local Storage

Draft reports, captures, settings, and optional credentials are stored locally using Chrome extension storage. Screenshot2Bug does not operate its own external server for storing this data.

## Third-Party Services

Screenshot2Bug only sends data to third-party services when you configure and use those integrations:

- OpenAI: report context may be sent to OpenAI to generate an AI-assisted bug report.
- GitHub: report content may be sent to GitHub to create an issue in the repository you select.
- Cloudinary: screenshots may be uploaded to Cloudinary so they can be embedded in GitHub issues.

Use of these services is optional and depends on settings you provide.

## Data Sharing

Screenshot2Bug does not sell user data. Screenshot2Bug does not share user data with third parties for advertising or marketing. Data is only shared with the optional services listed above when you choose to use those integrations.

## Remote Code

Screenshot2Bug does not load or execute remotely hosted JavaScript. Extension code is packaged with the Chrome extension.

## Data Retention

Draft reports and captured evidence remain in local Chrome extension storage until you remove the extension, clear extension storage, or overwrite/export reports through the extension’s normal workflows. Data sent to optional third-party services is retained according to those services’ own policies.

## Contact

For privacy questions or support, contact the publisher at the email address listed on the Chrome Web Store listing.
